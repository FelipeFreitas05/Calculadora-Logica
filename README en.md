LogicApex - Boolean Logic Calculator

LogicApex is an interactive and educational boolean logic calculator, created with HTML, Tailwind CSS, and pure JavaScript. It allows users to visualize the results of logic gates (AND, OR, NOT, etc.) in real-time, as well as build and analyze complex boolean equations with a step-by-step truth table.
